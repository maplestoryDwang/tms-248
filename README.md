## 说明
### 感恩
感恩所有愿意分享和热爱Maplestory的朋友，本仓库源地址：https://github.com/rage123450/TMS246-Server

### 项目说明
1. TMS-248.1的端，两年前在网上找的
2. 基于swordie架构的TMS端
3. jdk17
4. 数据库为：mariadb
5. 启动方式可以看看其他TMS启动方式

### 为什么要做这个项目
1. 虽然TMS245的魔森端已经非常优秀，但是我还是喜欢swordie的项目架构多一点，重构项目是一个大工程正好看到有这么一个项目，尝试跑了一下发现能正常启动。
2. 之前在研究GMS214，虽然也足够优秀，但是游戏分辨率让我头疼，4K屏幕看的非常难受，又不想HOOK修改内存，所以想选择一个高版本的进行修复
3. 当前已经有很多优秀的端进行参考。只凭借静态调试已经完全满足修复。

### 额外说明
1. 注意一下dat数据的存储路径和文件名，否则容易闪退
2. 项目缺失很多脚本，需要修复。
3. 当前还是以GMS214为主。修复完的功能如果这边可以用就merge到这边。

### 快速开始
1. 确保已安装JDK 17或更高版本
2. 安装并配置MariaDB数据库
3. 导入项目所需的数据库文件
4. 配置服务器设置
5. 启动服务器
